# package-lock.json

* package-lock.json文件解析

* name：项目的名称

* version：项目的版本

* lockfileVersion：lock文件的版本

* requires：使用requires来跟着模块的依赖关系

* dependencies：项目的依赖
  * 当前项目依赖axios，但是axios依赖followredireacts
  * axios中的属性如下
    * version表示实际安装的axios的版本
    * resolved用来记录下载的地址，registry仓库中的位置
    * requires记录当前模块的依赖
    * integrity用来从缓存中获取索引，再通过索引去获取压缩包文件
