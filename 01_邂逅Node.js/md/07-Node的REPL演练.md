# Node的REPL

- 什么是REPL呢？感觉挺高大上
  + REPL是Read-Eval-Print Loop的简称，翻译为“读取-求值-输出”循环；
  + REPL是一个简单的、交互式的编程环境；

- 事实上，我们浏览器的console就可以看成一个REPL。

- Node也给我们提供了一个REPL环境，我们可以在其中演练简单的代码。
